Overview

This site contains examples of accessible navigation menus.

Caveats

When reviewing these pages, please keep the following aspects in mind:

  • These menus have been optimized for accessibility. Some further testing with other accessibility experts and assistive technology users is planned, so some features may change.
  • The site contents are currently dynamically generated to provide placeholder text. It is fairly easy to change the pages and headings on a page in case more relevant categories are needed.
  • The pages and menus are not currently designed to be responsive as of yet. Performance on smaller screens will have issues.
  • Many stylistic aspects of the menus can be changed. This includes colors, spacing, icons, etc. Current colors do follow SU branding while allowing for accessibility.
  • These examples are for primary site navigation, meaning the main navigation typically at the top of a web page. The placement and styling of any secondary navigation can be discussed. The current secondary navigation seen in the right column of some pages is merely a placeholder for now.
  • All pages do have Breadcrumb navigation when appropriate. This is recommended for accessibility purposes.
